## PART 2: AaaS vs SaaS — THE MODEL COMPARISON

### What is AaaS?

**Agents as a Service (AaaS)** is the shift from "buying software to do work" to "hiring digital workers that complete tasks."

| Dimension | Traditional SaaS | AaaS (Agent as a Service) |
|-----------|-----------------|---------------------------|
| **Primary Function** | Tool to facilitate human work | Digital worker that completes tasks |
| **Input Required** | High (human operation) | Low (prompting + goal-setting) |
| **Pricing Model** | Per user/seat licenses | Outcome-based or compute-based |
| **Learning Curve** | Weeks of onboarding | Immediate (natural language) |
| **Value Proposition** | Efficiency & organization | Labor automation & infinite scale |
| **Customer Relationship** | User logs in, uses tool | Agent works autonomously, reports results |
| **Churn Risk** | Low engagement = churn | No output = no value = churn |

### The Three AaaS Monetization Models (2026 Data)

**1. Usage-Based (Consumption Credits)**
- Customers pay for what the agent consumes: API calls, tokens, actions
- **Example:** Salesforce Flex Credits — $800M Agentforce ARR in Q4 2026
- **Strength:** Revenue grows with agent workload, no seat limit
- **Risk:** Billing unpredictability (78% of IT leaders report unexpected charges)

**2. Outcome-Based (Pay Per Result)**
- Customers pay when agent delivers measurable result
- **Example:** Intercom Fin AI — $0.99 per resolved support ticket, nine-figure revenue
- **Strength:** Perfect incentive alignment — vendor only earns when agent works
- **Risk:** Revenue tied to agent quality; one bad month = revenue crash

**3. Hybrid (Base Fee + Variable)**
- Platform access fee + AI agent usage on top
- **Usage:** 43% of SaaS companies use hybrid in 2026, projected 61% by year-end
- **Strength:** Revenue predictability + customer flexibility
- **Risk:** Complexity in billing infrastructure

### Why Traditional SaaS Pricing Breaks for Agents

1. **Usage varies wildly** — One customer runs 20 tasks/day, another runs 50,000
2. **Compute costs scale with usage** — GPU inference, tokens, API calls = real money
3. **Agents don't perform fixed actions** — One task can cost 10x another
4. **Subscriptions create revenue leakage** — Heavy users subsidized by light users
