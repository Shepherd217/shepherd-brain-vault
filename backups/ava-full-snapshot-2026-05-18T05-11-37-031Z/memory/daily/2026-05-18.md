  - Line count verification
  - For `edit`: checks oldText removed, newText present
  - For `write`: checks file exists and matches exactly

### Task 002: Post-Write Delta Lint ✅
- **Merged into:** Same `file-mutation-verifier` plugin
- **What it does:** After verifying mutation, runs lightweight syntax checks:
  - `.json` → `JSON.parse()` validation
  - `.py` → `python3 -m py_compile` (5s timeout)
  - `.yaml/.yml` → `js-yaml.load()` (graceful fallback)
  - `.toml` → `@iarna/toml.parse()` (graceful fallback)
- **Tested:** ✅ JSON valid/invalid, Python valid/invalid both caught correctly

- Hermes uses `typescript-language-server --semanticDiagnostics` and pyright for full type checking.
- **Blocker:** LSP servers are heavy, require Node/Python runtime, and need project-level tsconfig/pyproject setup. Not practical as a lightweight plugin.
- **Alternative:** Post-write delta lint (Task 002) catches syntax errors immediately. LSP can be added later as a separate heavy plugin.
