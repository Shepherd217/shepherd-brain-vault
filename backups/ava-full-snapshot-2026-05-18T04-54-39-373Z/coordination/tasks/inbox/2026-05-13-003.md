## Bugs Found & Fixed

### BUG #1: Race Condition (Duplicate Claims)
**Issue:** Two agents could claim the same task. Second claim overwrote first.
**Fix:** Added guard in `claim` command — rejects if task already claimed by another agent.
**File:** `dispatch.py` — claim handler

### BUG #2: Corrupted Registry Silent Failure
**Issue:** Corrupted `registry.json` caused silent failure — no agents shown.
**Fix:** `load_json()` now detects corruption, prints warning, and recreates from defaults.
**File:** `dispatch.py` — load_json function

### BUG #3: Missing Signals Directory
**Issue:** `touch()` failed if parent `signals/` directory was missing.
**Fix:** `touch()` now creates parent directories with `mkdir(parents=True, exist_ok=True)`.
**File:** `dispatch.py` — touch function
