# Lazy-Deps / Debloating

Install heavy backends only on first use, not upfront.

## What Done Looks Like

- [ ] Tag each skill with `lazy_deps: ["pkg1", "pkg2"]` in SKILL.md frontmatter
- [ ] On skill load, check if deps installed. If not, install them.
- [ ] Cache installed state in `.clawdeps/` or `~/.openclaw/.lazydeps/`
- [ ] Workspace startup skips heavy imports unless needed

## Why This Matters

Install is getting heavy. Every new skill adds dependencies. Nathan shouldn't wait 30 seconds for imports he doesn't use.

## Files to Touch

- Skill loader (`skills/__init__.py` or equivalent)
- Each `SKILL.md` — add `lazy_deps` metadata
- Workspace bootstrap scripts
