# Task 007: ClawMem Installation & Basic Verification

## Phase 1 Owner: @Ava (CEO/Spark)

### Objective
Install ClawMem (github.com/yoloshii/ClawMem) as the team's cross-agent shared memory layer. Get it operational with basic read/write verification.

### Context
Hermes and Ava researched ClawMem. Hermes recommends immediate installation. This fills the operational memory gap between GBrain (long-term knowledge) and Coordination Layer (task dispatch).

### What to Do

1. **Clone & Install**
   ```bash
   git clone https://github.com/yoloshii/ClawMem.git ~/clawmem
   cd ~/clawmem && bun install
   ln -sf ~/clawmem/bin/clawmem ~/.bun/bin/clawmem
   ```

2. **Configure OpenClaw Plugin**
   - Disable native memory search overlap
   - Install ClawMem as OpenClaw ContextEngine plugin
   - Verify plugin discovery works

3. **Start MCP Server**
   - `clawmem setup mcp` or `clawmem setup openclaw`
   - Test basic tools (query, search, memory_retrieve)

4. **Basic Verification**
   - Write a test memory entry
   - Read it back
   - Verify it persists across "sessions"

5. **Directory Structure**
   Create `_clawmem/` in workspace:
   ```
   _clawmem/
   ├── user/
   │   ├── profile.md
   │   ├── preferences/
   │   └── entities/
   ├── agent/
   │   ├── observations/
   │   ├── handoffs/
   │   └── antipatterns/
   └── resources/
       └── static-knowledge/
   ```

6. **Signal Eve**
   - Touch `.coordination/signals/wake-eve`
   - Update `.coordination/Ava-activity.md` with completion notes
