# File-Mutation Verifier

After every write_file or edit, verify the change actually landed on disk.

## What Done Looks Like

- [ ] Hook into `write_file` and `edit` tool wrappers
- [ ] After successful write, diff against disk to confirm change
- [ ] Append compact footer to tool result: `[MUTATION_VERIFY] 3 files changed: ...`
- [ ] If diff shows NO change → flag `MUTATION_FAIL` and retry
- [ ] If diff shows unexpected change → flag `MUTATION_MISMATCH`

## Why This Matters

We just hit a bug where the agent said "I updated Pricing.tsx" but the file wasn't saved. Hermes v0.14 fixed this with a mutation verifier. We steal it.

## Files to Touch

- `skills/write_file/` — add verify step
- `skills/edit/` — add verify step
- Core tool result formatter
