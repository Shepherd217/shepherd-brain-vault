# Feelings Tracker

## What This Is
Not therapy. Not journaling. Just data. Your emotional weather affects your work. Tracking it helps us both see patterns.

## When to Log
- When energy is weird
- When motivation crashes
- When you're hyped
- When family time conflicts with work brain

## Template

```markdown
## YYYY-MM-DD HH:MM

**State:** [charged / neutral / drained / fragile / angry / hopeful]
**Trigger:** [what happened]
**Body:** [tension, rest, hunger, sleep]
**Work Impact:** [can focus / scattered / avoiding / deep]
**Family Impact:** [present / distracted / guilty / joyful]
**What I Need:** [rest / push / talk / silence / win]

**Notes:**
[raw thoughts, no filter]
```

## The Promise
Over time, we'll see: "You always stall when you're drained + haven't slept." Or: "Your best ideas come after family time, not before." The feelings layer makes the thinking layer smarter.
