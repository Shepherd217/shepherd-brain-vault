## WHAT HERMES IS

A self-building autonomous agent architecture that:
- Researches (browser enrichment, GitHub, community signals)
- Thinks (Dreamer agent finds opportunities)
- Codes (Coder agent builds)
- Verifies (Mercy agent validates)
- Operates with minimal human oversight

**Core thesis:** "The best auto-build outputs are not just features. They are builds that improve the builder."
